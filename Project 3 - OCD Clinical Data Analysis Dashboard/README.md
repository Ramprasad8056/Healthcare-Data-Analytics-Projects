# 🧠 OCD Clinical Data Analysis Dashboard

## 📊 Project Overview
This project analyzes a dataset of 1,500 OCD patients to identify key clinical patterns, severity distribution, comorbid conditions, and treatment insights.

The analysis was performed using:
- SQL (MySQL) for data extraction and transformation
- Power BI for interactive visualization and dashboard creation

---

## 🎯 Objectives
- Analyze severity distribution of OCD patients
- Study relationship between depression and severity
- Identify trends between symptom duration and severity
- Evaluate demographic patterns (gender, marital status, ethnicity)
- Explore medication usage and its association with severity

---

## 🛠 Tools Used
- MySQL
- Power BI
- Power Query (Data Transformation)

---


## 📈 Key Insights
- Higher symptom duration is associated with increased severity
- Depression prevalence is higher in severe and extreme categories
- Severity distribution shows significant clustering in moderate to severe ranges
- Gender distribution is nearly balanced across severity levels

---

## 🧩 Dashboard Features
- Interactive dashboard with KPI cards, slicers, and dynamic visuals for real-time analysis
- Includes demographic, clinical severity, and treatment-based insights
- Uses scatter, bar, and donut charts to explore patterns and relationships
- Enables multi-dimensional filtering by gender, severity, medication, and ethnicity

---

## 🧠 Conclusion
- Higher symptom duration is associated with increased OCD severity
- Depression is more prevalent in severe and extreme cases
- Most patients fall within moderate to severe severity categories
- Treatment and severity patterns vary across medication groups
