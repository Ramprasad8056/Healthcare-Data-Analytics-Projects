# 🚦 Road Accident Analysis – India (2020)
##📊 Project Overview
This project analyzes road accident data across major Indian cities to identify accident trends, high-risk locations, fatality rates, and major contributing causes.
The goal is to transform raw accident data into meaningful insights using Excel, SQL, and Power BI and present them through an interactive dashboard.

## 🎯 Objective
Identify cities with the highest accident occurrences
Analyze fatality and injury trends
Determine major accident causes
Visualize geographic accident distribution
Build an interactive analytical dashboard

## 🛠 Tools & Technologies
Excel – Data cleaning and preliminary analysis
SQL – Querying and data aggregation
Power BI – Data visualization and dashboard development

## 📈 Key Insights
Major metropolitan cities recorded the highest accident volumes.
Traffic violations and road infrastructure issues were significant contributors to accidents.
A large proportion of accidents resulted in injuries, while a smaller percentage resulted in fatalities.
Certain accident causes repeatedly appeared among the top contributors across cities.
Geographic visualization helped identify accident hotspots across India.

## 📊 Dashboard Features
The Power BI dashboard includes:
Total Accidents, Injuries, Deaths, and Fatality Rate KPIs
Interactive metric selector (Accidents / Deaths / Injuries)
Map visualization of accident distribution across major cities
Top 10 cities with highest accident volume

## 📌 Conclusion
This project demonstrates data cleaning, querying, exploratory analysis, and dashboard development skills. By combining SQL analysis with Power BI visualization, the project highlights meaningful patterns in road accident data and provides a clear analytical view of accident trends and causes.
