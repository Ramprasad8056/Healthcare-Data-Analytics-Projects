## Objective
Analyze global COVID-19 clinical trial data to identify patterns in status, phases, funding, enrollment, and research trends.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Google Colab
## Key Insights
- Majority of trials were recruiting or completed.
- Phase 2 and Phase 3 trials dominated.
- Countries with advanced research infrastructure led activity.
- Government and industry funding were major contributors.
- Trial activity surged during peak pandemic years.

## Conclusion
This project demonstrates exploratory data analysis skills in healthcare research data.
